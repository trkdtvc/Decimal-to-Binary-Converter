# Decimal-to-Binary-Converter

A JavaScript-based decimal to binary converter with dynamic output and a visual call stack animation.

## Features
- Converts decimal input to binary output
- Updates the result dynamically in the UI
- Includes a visual call stack animation to illustrate the conversion process

## Tech Stack
- JavaScript
- HTML
- CSS

## Purpose
Built as a practice project to strengthen understanding of number conversion and recursion concepts, with a simple UI and a visualization of the call stack.
